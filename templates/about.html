About Database Studio

DB Studio is a modern web-based database management tool built with Flask, Tailwind CSS, and Python.

Features

  • Connect to MySQL databases remotely
  • SQL Editor with autocomplete and query history
  • Inline edit, insert, and delete table rows
  • Import and export tables (CSV / SQL)
  • ER diagrams and table relation graphs
  • Audit logs and role-based permissions
  • Multi-database switch and dark mode UI
